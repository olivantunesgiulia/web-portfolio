# Portfólio Web | Giulia Antunes

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript com o objetivo de apresentar meus projetos, habilidades e trajetória como desenvolvedora fullstack em formação.

O projeto foi criado com foco em:
- design moderno
- organização visual
- experiência do usuário
- responsividade
- interatividade com JavaScript

---

# Funcionalidades

- Navegação entre seções com scroll suave
- Navbar dinâmica com destaque da seção ativa
- Animações de fade-in ao scroll
- Cards de projetos personalizados
- Páginas individuais para cada projeto
- Formulário de contato com feedback visual
- Botão de voltar ao topo
- Layout responsivo
- Integração com GitHub e LinkedIn
- Download/visualização de currículo em PDF

# Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>

```

2. Abra a pasta do projeto:

```bash
cd projeto-web-portfolio

```

3. Execute o arquivo home.html no navegador.
